# Project Documentation

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/alokag/doc-help.git
   cd doc-help
   ```
2. Install the necessary dependencies:
   ```bash
   npm install
   ```

## How to Run Locally
1. Start the local server:
   ```bash
   npm start
   ```
2. Open your browser and navigate to `http://localhost:3000` to view the application.

## Deployment on Render
1. Log in to your Render account.
2. Click on "New" then select "Web Service".
3. Connect your GitHub account and select this repository.
4. Set the environment variables as required for your application.
5. Click on "Create Web Service" to deploy your application.

## Setting Webhook URL for Meta WhatsApp Cloud API
1. Go to the Meta Developers Portal.
2. Open your app and navigate to the "Webhook" section.
3. Enter the URL where your application is running (e.g., `https://yourapp.render.com/webhook`).
4. Choose the events you want to subscribe to and click on the "Subscribe" button.
